# Text to Speech Converter

#### Веб-приложение для преобразования текста в живую речь. Проект демонстрирует навыки работы с браузерным API, асинхронностью (события загрузки голосов) и современными методами верстки.

# Стек технологий
#### JavaScript (ES6+): Манипуляция DOM, работа с объектами SpeechSynthesis и SpeechSynthesisUtterance.
#### HTML5: Семантическая разметка.
#### CSS3: Flexbox для позиционирования, CSS-переменные, медиа-запросы для адаптивности и продвинутые визуальные эффекты.

# Основной функционал
#### Синтез речи в реальном времени: Использование Web Speech API для мгновенной озвучки введенного текста.
#### Выбор голосов: Динамическая подгрузка списка доступных системных голосов (зависит от ОС и браузера пользователя).
#### Адаптивный дизайн: Интерфейс полностью оптимизирован под мобильные устройства и десктопы.
#### Glassmorphism UI: Стильное оформление с использованием эффектов размытия (backdrop-filter), градиентов и мягких теней.

# Структура проекта
#### index.html — Структура приложения.
#### style.css — Стилизация и адаптивность.
#### script.js — Логика работы с API и обработка событий.
